# 📚 Документация проекта — Справочник технологий

Здесь собраны простые объяснения всех технологий и библиотек, которые используются
в приложении «Передача времени приёма лекарств». Каждая статья написана так,
чтобы разобраться мог даже человек, который никогда раньше не программировал.

---

## Быстрая карта проекта

```
Пользователь нажимает кнопку
        │
        ▼
   Fragment (экран)          ← UI-компоненты (раздел 06)
        │                       Android (раздел 02)
        ▼
   ViewModel                 ← MVVM (раздел 03)
        │                       Корутины и Flow (раздел 07)
        ▼
   Repository
    /        \
DAO (Room)   BluetoothManager
    │              │
База данных    Arduino        ← Room (раздел 04)
(раздел 04)    по Bluetooth   ← Bluetooth (раздел 05)
```
